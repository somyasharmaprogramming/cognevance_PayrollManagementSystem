# Employee Payroll Management System

## Features
- Employee Registration
- Attendance Management
- Payroll Processing
- Salary Calculation
- JDBC Connectivity
- MySQL Database
- Multithreading
- Exception Handling

## Technologies Used
- Java
- JDBC
- MySQL

## How to Run
1. Install MySQL.
2. Create database using payroll.sql.
3. Add MySQL JDBC Driver.
4. Compile Java Files.
5. Run Main.java.
